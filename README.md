# vlang-sample-box

Vlang sample and snippet code.

# Install vlang

```
$ brew install vlang
$ v -version
V 0.2.4 b72a2de
```

# Build and run

```
# build
$ v main.v

# run
$ ./main

# build and run
$ v run main.v

# compile ( and cross-compilation )
v . -o /tmp/app
# v . -o /tmp/app -os macos
# v . -o /tmp/app -os linux
# v . -o /tmp/app.exe -os windows
```

# Concerns

- Handling for nullable field in json.
- Optional fields in struct.
